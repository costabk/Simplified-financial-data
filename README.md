# 📈 Global Stock Comparison Dashboard

A professional-grade financial analytics tool built with Python. This dashboard allows users to compare the financial health and stock performance of major global companies side-by-side using real-time data.

## 🛠️ Features
- **Real-time Data:** Fetches live financial metrics via Yahoo Finance API.
- **Side-by-Side Comparison:** Compare two tickers (e.g., AAPL vs MSFT) simultaneously.
- **Key Metrics:** Visualize Total Revenue, Net Income, and Revenue Growth (YoY).
- **Normalized Performance Chart:** Stock prices are normalized to 100% to allow a fair growth comparison regardless of share price.
- **Responsive Design:** Built with Streamlit for a seamless experience on both desktop and mobile.

## 🧰 Tech Stack
- **Language:** Python 3.13
- **Framework:** [Streamlit](https://streamlit.io/)
- **Data Source:** [yfinance](https://pypi.org/project/yfinance/)
- **Data Manipulation:** Pandas
